# Role
Act as an expert Full-stack Engineer and UI/UX Designer specializing in Astro, Tailwind CSS, and academic portfolios.

# Task
Build a high-performance, minimalist personal academic website for a Computer Science PhD student. The site must be optimized for SEO, accessibility, and technical readability.

# 1. Tech Stack & Architecture
- **Framework:** Astro (Static Site Generation).
- **Styling:** Tailwind CSS + @tailwindcss/typography.
- **Content:** Markdown/MDX-based workflow for Publications and Write-ups.
- **Math:** KaTeX integration via Remark-Math and Rehype-KaTeX for server-side LaTeX rendering.
- **Deployment:** Optimized for GitHub Pages.

# 2. Visual Aesthetic & Appearance (CRITICAL)
Implement a dual-theme system with a smooth CSS transition:

## Dark Mode (Noir/Fintech Style)
- **Background:** Deep True Black (#000000).
- **Accent:** "Volt" Green (#D4FF80) for primary CTA buttons and key header highlights.
- **Typography:** High-contrast white for headings; muted silver for body.
- **Image Styling:** Implement a "Spotlight" effect using a radial-gradient mask on the profile and featured project images so they fade into the black background.

## Light Mode (Glassmorphic/SaaS Style)
- **Background:** Clean white with large-radius, low-opacity pastel gradients (soft lavender/blue).
- **Components:** Use "Glassmorphism"—semi-translucent white backgrounds, 1px subtle borders, and `backdrop-blur`.
- **Typography:** Dark slate for high readability.

# 3. Site Structure & Pages
- **Home:** Bio, professional headshot (with spotlight effect), social icons (GitHub, LinkedIn, Google Scholar), and a vertical "News" timeline.
- **Research:** Overview of research interests and active project cards.
- **Publications:** - Reverse-chronological list with category filtering (Conference, Journal, Workshop).
    - Metadata: Title, Authors (Bold my name), Venue, Year.
    - Action Buttons: [PDF], [Code], [Project Page].
    - **BibTeX Modal:** A popup component to copy citations to the clipboard.
- **Write-ups (Technical Blog):**
    - A 3-column grid layout for post previews.
    - Full MDX support for:
        - Inline and block LaTeX equations (e.g., $$\mathcal{L}(\theta)$$).
        - Syntax-highlighted code blocks (using Shiki).
        - Optimized image insertions.
        - Auto-generated Table of Contents (ToC) for headings.
- **CV:** A summary page with a prominent "Download PDF" button.

# 4. Global Components
- **Navigation:** Sticky Glassmorphic header with a theme toggle (Sun/Moon icons).
- **SEO:** Metadata manager for OpenGraph tags and Google Scholar indexing.
- **Footer:** Simple copyright and "Built with Astro" credits.
